# java-baseball
숫자 야구게임 미션을 진행하는 저장소

# 기능 메소드 목록
 1. createNumber() 
  - 기본적으로 1 ~ 9까지의 숫자를 중복없이 생성해주는 메소드
  
 2. inputNumber()
  - User에게 3가지 숫자를 입력받는 메소드
   2-1. resave()
    - User에게 입력받은 String을 다시 int 배열로 변환해주는 메소드
